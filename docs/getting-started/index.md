# Getting Started with RxInfer Client

Welcome to the RxInfer Client documentation! This guide will help you get started with using our TypeScript SDK to interact with RxInferServer.

## Installation

First, install the package using npm:

```bash
npm install @lazydynamics/rxinfer-client
```

## Overview

The RxInfer Client provides a type-safe way to interact with RxInferServer. It automatically generates TypeScript types and client code from the OpenAPI specification, ensuring type safety and better developer experience.
